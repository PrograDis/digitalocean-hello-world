Hola desde DigitalOcean

Esta landing Hello World prueba el flujo de despliegue para un sitio estático en DigitalOcean App Platform, aprovechando la CDN global y certificados TLS automáticos.

Clona el repositorio, ejecuta doctl apps create --spec app.yaml y tendrás la aplicación online en minutos.

Consulta la documentación oficial para personalizar redirecciones, caché y conexiones a APIs externas desde docs.digitalocean.com.
